Команды Git
Команда Описание
git init инициализировать репозиторий в текущем каталоге
git init dir инициализировать репозиторий в каталоге `dir`
git clone url клонирует репозиторий `url` в локальную папку
git add file добавляет изменения `file` в раздел проиндексированных файлов
git add . добавляет изменения всех файлов в каталоге в проиндексированные файлы
git commit -m "comment" добавляет файл в коммит `comment`
git commit -a -m "comment" автоматически добавить все измененные файлы в коммит `comment`
git push url отправить данные в удалённый репозиторий `url`
git status -
git log -
git branch branch_name -
git checkout branch_name -
git checkout -b branch_name -
git remote -
